# Food-Delivery
backend project
A modern food delivery web app built with React and Vite.

Features:
1.User authentication
2.Dynamic menu browsing
3.Cart management
4.Fully responsive design

Tech Stack:
Frontend: React, Vite
Backend: Node.js, Express
Database: MongoDB

To run the project:
1.Clone the repo
2.Install dependencies: npm install
3.Start the server: npm run dev
